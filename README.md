# customer-shopping-behavior-analysis


## 📌 Project Overview

This project analyzes customer shopping behavior using **Python**, **PostgreSQL**, and **Power BI** to uncover valuable business insights.

The analysis explores customer demographics, purchasing behavior, discounts, product performance, customer segmentation, seasonal trends, and payment preferences.

The project follows a complete analytics workflow:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Queries
- Interactive Power BI Dashboard
- Business Insights & Recommendations
